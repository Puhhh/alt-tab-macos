# AltTabWindows

Небольшое нативное macOS-приложение на Swift, которое переключает не только между приложениями, а между отдельными окнами по `Option + Tab`.

## Что уже есть

- глобальный хоткей `Option + Tab`
- HUD-оверлей со списком окон
- фокусировка выбранного окна через Accessibility API
- главное окно с подсказкой по разрешениям

## Как запустить

1. Установить полный Xcode.
2. Открыть `AltTabWindows.xcodeproj`.
3. Выбрать target `AltTabWindows`.
4. Запустить приложение.
5. Выдать доступ в `System Settings > Privacy & Security > Accessibility`.

## Как пользоваться

- удерживай `Option`
- нажимай `Tab`, чтобы двигаться вперёд по окнам
- отпусти `Option`, чтобы активировать выбранное окно
- нажми `Esc`, чтобы закрыть оверлей без переключения

## Ограничения MVP

- проект рассчитан на современные версии macOS и Apple Silicon
- переключение работает через Accessibility, поэтому без разрешения оно не будет доступно
- порядок окон основан на снимке текущих видимых окон с лёгкой локальной историей, а не на полном системном MRU

## Примечание по сборке в этой среде

В текущем окружении доступен только `CommandLineTools`, без полного Xcode, поэтому полноценную сборку `.app` через `xcodebuild` здесь проверить нельзя. Сам проект и исходники подготовлены так, чтобы их можно было открыть и собрать в Xcode на macOS.
